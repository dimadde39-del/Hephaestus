# Demo Screenshot Pack

These images are curated terminal captures from the working local CLI. They are
intended for the README, docs, and the first soft reveal posts.

- `release-plan-demo.png`: end-to-end release planning command.
- `repo-inspect-demo.png`: read-only repo inspection.
- `pareto-demo.png`: Pareto tradeoff comparison.
- `qubo-demo.png`: QUBO problem inspection.
- `explain-demo.png`: decision summary from a saved optimizer run.

The screenshots use real command output captured from this repository. They are
cropped for readability, so the full CLI output may continue beyond the image.
